# Instrucciones para ejecutar los proyectos

## 1. Requisitos
- Java 17+
- Maven
- MySQL (opcional según el proyecto)
- VS Code con extensión Spring Boot Dashboard

## 2. Paso a paso
1. Clonar el repositorio:
   git clone https://github.com/lauramendoza/backend-portfolio

2. Ingresar al proyecto que quieras ejecutar:
   cd proyectos/proyecto-01

3. Ejecutar:
   mvn spring-boot:run

4. Abrir en el navegador:
   http://localhost:8080
